Rekursi Magic: 1 + 2 + ... + n

Base Case · Recursive Case · Call Stack (LIFO)

Fungsi memanggil dirinya sendiri layaknya forward message berantai. Tentukan Base Case (n == 1) dan Recursive Case (n + sum(n-1)).

🧠 Eksplorasi Rekursi
Hasil Akhir: —
sum(6) = 6 + 5 + 4 + 3 + 2 + 1 = ?
Kode Rekursif 
function sum(n) {
    if (n == 1) return 1;        // ✨ Base Case
    else return n + sum(n - 1);  // 🔁 Recursive Case
}

Base Case (n == 1) → berhenti. Recursive: pecah masalah menjadi n + sum(n-1)
